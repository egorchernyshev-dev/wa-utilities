# ym-counters-to-table

Функционал для выгрузки ифнормации по всем счетчикам с открытой страницы списка счетчиков Яндекс Метрики. 

1. Кликаем на странице списка счетчика "Показать еще" до тех пор пока эта кнопка перестанет появляться
2. В консоли запускаем скприт из index.js 
3. Кликаем "копировать" на появившемся сообщении. 
4. Вставляем в таблицу.
