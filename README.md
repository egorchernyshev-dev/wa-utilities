# wa-utilities
Репозиторий для утилит ВА
## ym-counters-to-table
Функционал для выгрузки иформации по всем счетчикам с открытой страницы списка счетчиков Яндекс Метрики. 

1. Кликаем на странице списка счетчика "Показать еще" до тех пор пока эта кнопка перестанет появляться
2. В консоли запускаем скприт из index.js 
3. Кликаем "копировать" на появившемся сообщении. 
4. Вставляем в таблицу. 

## ym-check-filter
Утилита для проверки фильтров ЯМ на дубликаты и на наличие минимально необходимых. 
Для работы данного коснольного скрипта откройте страницу фильтров ЯМ, на ней - консолью. Перейдите в контекст "settings". А уже из него в вставьте и запустите скрипт в консоли.
